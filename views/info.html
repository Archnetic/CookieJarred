Info:

Cookie Jarred is an open source extension that is activly being updated.

Cookie Jarred runs everytime a new website is loaded. It will search for the cookie acception layer if by HTML class and will attempt to remove it. If the cookie layer does not get removed pleas submit a Github issue via the "report a cookie" button on the main page. When reporting a cookie please include the link of the website where it occured.

In some cases the cookie layer might be removed between 1 - 2 seconds after the website loads.
